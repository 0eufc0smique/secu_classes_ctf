## Security Learning & CTF Repository

Repo originally based on https://github.com/Crypto-Cat/CTF and adjusted to my needs.

**Note**: Not all CTF writeups and scripts are my own work. Most of the original content comes from CryptoCat.

## Directory Structure

```
secu_classes_ctf/
├── learning/           # Educational content
│   ├── courses/        # Structured courses and tutorials
│   ├── practice/       # Practice challenges by platform
│   └── books/          # Book-related content
├── events/             # CTF competitions
│   ├── mine/          # My CTF participations
│   │   └── 2024/     # ecw_2024, grehack_2024, hero_ctf_2024
│   └── cryptocat/     # CryptoCat's CTF writeups
│       ├── 2024/     # cyber_apocalypse_24
│       ├── 2023/     # intigriti, cyber_apocalypse, sekai, etc.
│       ├── 2022/     # angstrom, pico, nahamcon, etc.
│       ├── 2021/     # csaw, hacktivitycon, killerqueen, etc.
│       └── archive/  # Misc/bonus content
├── resources/         # Reference materials
│   ├── cheatsheets/   # Security cheatsheets
│   ├── libs/          # Libraries (GLIBC, etc.)
│   └── certs/         # Certificates
└── tools/             # Security tools and scripts
    ├── pwn/           # Binary exploitation scripts
    ├── pentest/       # Pentesting scripts
    ├── kernel/        # Kernel utilities
    └── misc/          # Miscellaneous tools
```

## Organization Philosophy

- **learning/**: Long-term skill development (courses, practice platforms)
- **events/**: Time-bound CTF competitions
  - **mine/**: My personal CTF participations
  - **cryptocat/**: CryptoCat's writeups organized by year
- **resources/**: Reference materials and dependencies
- **tools/**: Reusable scripts and utilities
